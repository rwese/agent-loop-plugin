# Custom Continuation Prompt Template

You can create a custom continuation prompt by specifying the path in your configuration:

```json
{
  "continuationPromptFile": "/path/to/your/custom-prompt.txt"
}
```

## Available Placeholders

The following placeholders are supported:

- `{incompleteCount}` - Number of incomplete tasks
- `{todoList}` - Formatted list of pending tasks
- `{totalCount}` - Total number of tasks
- `{completedCount}` - Number of completed tasks

## Example Custom Prompt

```txt
[TASK CONTINUATION - {incompleteCount} tasks remaining]

Please continue working on the following tasks:

{todoList}

Next steps:
1. Pick ONE task from the list above
2. Complete it thoroughly
3. Update the todo status
4. Move to the next task

Remember: Work independently and don't ask for permission!
```

## Default Location

The default continuation prompt is located at:
`./prompts/continuation.txt`
